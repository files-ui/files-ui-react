<h1 align="center">Files ui - core</h1>

Base code for **Files UI** 


## Installation

@files-ui/react is available as an [npm package](https://www.npmjs.com/package/@files-ui/react).

```sh
// with npm
npm i @files-ui/core
```

```sh
// with yarn
yarn add @files-ui/core
```
